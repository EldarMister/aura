# Звуки озвучки «время вышло»

Сюда кладутся звуки, которые проигрываются на экране «Телевизор», когда у стола
заканчивается забронированное время (обратный отсчёт дошёл до 00:00:00).

| Файл | Когда играет |
| --- | --- |
| `table1.wav` | время вышло у **Стола 1** |
| `table2.wav` | время вышло у **Стола 2** |

Сейчас здесь лежат «тихие» заглушки, чтобы проект собирался. **Замени их своими
файлами, не меняя имена** (`table1.wav`, `table2.wav`). Поддерживается `.wav`.

После замены перезапусти бандлер с очисткой кэша ассетов:

```
npx expo start --clear
```
